## Script Guidelines

When adding Python scripts to this directory:

1. **Use proper script headers:**
   ```python
   #!/usr/bin/env python3
   """
   Description: What the script does
   
   Usage:
       python script-name.py [arguments]
       
   Dependencies:
       - package1
       - package2
   
   Author: Name
   Date: YYYY-MM-DD
   """
   ```

2. **Follow best practices:**
   - Use Python 3.6+ features
   - Include docstrings for functions and classes
   - Implement proper argument parsing with argparse
   - Add error handling with try/except
   - Use type hints where appropriate
   - Follow PEP 8 style guidelines

3. **Manage dependencies:**
   - Create requirements.txt if external packages needed
   - Use virtual environments for testing
   - Document minimum Python version required

4. **Document in this README:**
   - Script name and purpose
   - Usage examples with command-line arguments
   - Required and optional parameters
   - Dependencies and installation instructions
   - Configuration requirements
